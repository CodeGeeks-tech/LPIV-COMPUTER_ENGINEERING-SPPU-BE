üîπ 1. recursive_fibonacci(n) ‚Äì Recursive Approach
def recursive_fibonacci(n): 
    if n <= 1: 
        return n 
    else: 
        return recursive_fibonacci(n-1) + recursive_fibonacci(n-2)

‚úî Explanation:

The Fibonacci sequence is defined as:
F(0) = 0, F(1) = 1
F(n) = F(n-1) + F(n-2) for n > 1

This function directly implements that mathematical definition using recursion.

Base Case:
If n is 0 or 1 ‚Üí return n

Recursive Case:
Otherwise, call the function again for previous two values and return their sum.

‚úî Characteristics:

Easy to understand but slow, because many values are recomputed repeatedly.

Time Complexity: O(2‚Åø)

Space Complexity: O(n) (due to recursion stack)

üîπ 2. non_recursive_fibonacci(n) ‚Äì Iterative Approach
def non_recursive_fibonacci(n): 
    first = 0 
    second = 1 
    print(first)
    if n > 1:
        print(second)
    for i in range(2, n):
        third = first + second 
        print(third)
        first, second = second, third 

‚úî Explanation:

Starts by manually printing the first two Fibonacci numbers: 0 and 1.

If the user wants more than one term, it prints both.

Then it enters a loop from 2 up to n:

Computes the next Fibonacci number as:
third = first + second

Prints this number

Updates variables for the next iteration:
first ‚Üê second
second ‚Üê third

‚úî Characteristics:

Much faster than recursion.

Time Complexity: O(n)

Space Complexity: O(1) (only three variables used)
