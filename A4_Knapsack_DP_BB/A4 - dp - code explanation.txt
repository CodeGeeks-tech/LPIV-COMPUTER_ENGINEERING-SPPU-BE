def solve_knapsack_tabulation():


This function runs the whole knapsack program:

Takes input

Builds DP table

Computes maximum value

2Ô∏è‚É£ Taking number of items
n = int(input("Enter number of items: "))


User enters how many items are available.

3Ô∏è‚É£ Reading item values
val = []
print("Enter values of items:")
for i in range(n):
    val.append(int(input(f"Value of item {i+1}: ")))


val[i] stores profit of item i.

4Ô∏è‚É£ Reading item weights
wt = []
print("Enter weights of items:")
for i in range(n):
    wt.append(int(input(f"Weight of item {i+1}: ")))


wt[i] stores weight of item i.

5Ô∏è‚É£ Capacity of Knapsack
W = int(input("Enter maximum weight capacity: "))


This is the maximum weight the knapsack can hold.

6Ô∏è‚É£ DP Table Creation
dp = [[0] * (W + 1) for _ in range(n + 1)]

Meaning of dp[i][w]:

Maximum value possible using first i items with capacity w.

Table dimensions:

Rows = items + 1

Columns = weight capacity + 1

All initialized to 0.

7Ô∏è‚É£ Filling the DP Table (Main Logic)
for i in range(1, n + 1):
    for w in range(1, W + 1):


We fill row by row.

Case-1: Item cannot be included
if wt[i - 1] > w:
    dp[i][w] = dp[i - 1][w]


If item weight > current capacity,
‚û° We cannot include the item.
So take previous row value.

Case-2: Item can be included OR excluded
dp[i][w] = max(
    val[i - 1] + dp[i - 1][w - wt[i - 1]],   # include the item
    dp[i - 1][w]                             # exclude the item
)


We choose the better of:

‚úî Include item

Profit =

val[i-1] + dp[i-1][w - wt[i-1]]


Means:

Add item‚Äôs value

Reduce capacity

‚úî Exclude item

Profit =

dp[i-1][w]


We take the maximum ‚Üí optimal choice.

This is the essence of DP.

8Ô∏è‚É£ Final Answer
print("\nMaximum value using DP:", dp[n][W])


This cell contains:
best possible value using all n items with capacity W.

üß† CONCEPTUAL EXPLANATION (Write This in Exam)

This program solves the 0/1 Knapsack Problem using Dynamic Programming ‚Äì Tabulation.

dp[i][w] represents the maximum value that can be achieved using first i items with a knapsack capacity w.

The table is filled using two choices:

Exclude the item

Include the item (only if weight allows)

The value stored is the maximum of these two choices.

The final answer lies in dp[n][W].

Time Complexity:
O(nW)

Space Complexity:
O(nW)
